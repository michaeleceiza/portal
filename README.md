# Portal do Cliente — Eceiza Nunes Advogados Associados

Página pública do portal de acompanhamento processual do escritório.
O cliente recebe um link individual e secreto e, após confirmar o CPF/CNPJ
do titular, consulta o andamento dos seus processos.

Esta página não contém nenhum dado: o conteúdo é carregado em tempo real,
mediante o código secreto do link, e a segurança é garantida pelo servidor
(Supabase, com RLS). O conteúdo é publicado e atualizado exclusivamente
pelo advogado responsável.
